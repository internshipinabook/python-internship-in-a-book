# Datasets

This folder contains the datasets used throughout the Python Internship in a Book™ experience.

These datasets are designed for:
- hands-on learning,
- data cleaning practice,
- pandas exercises,
- visualization tasks,
- and internship-style mini projects.

## Included Datasets

- golden_fork_clean_final.csv
- golden_fork_menu.csv
- golden_fork_orders.csv
- golden_fork_orders_messy.csv

Learners are encouraged to:
- explore the datasets,
- modify analyses,
- create visualizations,
- and build additional portfolio projects.
